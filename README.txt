Алгоритм запуска программы:
Для начала работы программы запустите файл parsing.py, это позволит новостям обновиться и выдавать более свежие результаты.
Далее запустите ML.py, это обученная модель, готовая выдавать новости по их важности для бизнес-роли.
Запустите api.py, и для проверки можете запустить check.py, где с помощью запроса get вы получите релевантные новости на основе ваших бизнес-препочтений.
